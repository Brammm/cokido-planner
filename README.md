# Cokido Planner

[Cokido](https://cokido.org/) is a Belgian initiative to provide out-of-school childcare,
organised by parents, grandparents... A lot of the planning currently is done manually.
This is a tool that attempts to make that easier.

## Running locally

Note: Docker is required to be installed.

- Copy .env.example to .env, the defaults are sufficient.
- Run `docker compose up`
